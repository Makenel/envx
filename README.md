### üîß Basic Commands

Here are some basic commands to help you manage your environment variables:

- **Set an Environment Variable:**
  ```
  envx set VAR_NAME value
  ```

- **Get an Environment Variable:**
  ```
  envx get VAR_NAME
  ```

- **List All Variables:**
  ```
  envx list
  ```

- **Delete an Environment Variable:**
  ```
  envx delete VAR_NAME
  ```
